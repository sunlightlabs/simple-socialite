Hello world with simple-socialite:

This example makes use of extra extensions from drinks/socialite-extensions.

To include them in your build, add 'git://github.com/drinks/socialite.git#extensions' to EXTENSIONS_SOURCES in your settings file, and build like this:


            cake -e 'twitter-simple facebook-share linkedin linkedin-simple googleplus-simple hackernews tumblr-simple' build

Then run an http server in the root folder of simple-socialite and you're all set!
